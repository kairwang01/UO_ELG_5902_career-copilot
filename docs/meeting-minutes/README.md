# Meeting Minutes

This folder stores the minutes for all Team 8 meetings.

## Naming Convention

```
YYYY-MM-DD-<meeting-type>.md
```

Examples:
- `2026-05-10-weekly-sync.md`
- `2026-05-24-sprint-planning.md`
- `2026-06-07-sprint-retro.md`

## Template

```markdown
# <Meeting Type> — <Date>

**Time:** Sunday 8:00 PM EST
**Attendees:** Kair, Jingxuan, Xiaoyan, Xiaoyi, Jiaoyang
**Absent:** —
**Note-taker:** <name>

## Agenda
1. ...

## Discussion
- ...

## Decisions
- ...

## Action Items
| # | Action | Owner | Due |
|---|--------|-------|-----|

## Next Meeting
<date / time>
```

Minutes must be posted here within **24 hours** after the meeting and linked in Discord.
